# Food Mobile — Node.js + PWA scaffold

Run locally:

```bash
cd "c:\Users\usama hassan gurowa\CODING\food-mobile"
npm install
npm start
# open http://localhost:3000
```

Pages:
- `/` — menu + cart + place order
- `/contact.html` — contact numbers
- `/address.html` — physical address

API:
- `GET /api/menu` — returns menu
- `POST /api/orders` — create order (persisted to `orders.json`)
